\setlength{\absparsep}{18pt} % ajusta o espaçamento dos parágrafos do resumo
\begin{resumo}[Resumo]
	\SingleSpacing
	Comunicação Vehicle-to-Everything (V2X) requer garantias de latência determinísticas para aplicações de segurança, as quais não são fornecidas pelo Wi-Fi convencional baseado em CSMA/CA. Esta dissertação propõe o \textbf{DQ-WiFi}, uma estrutura MAC que combina reserva de banda periódica com escutas coordenadas para fornecer QoS determinístico em dispositivos Wi-Fi legados. O mecanismo central é uma Máquina de Estados Finitos (FSM) sincronizada por beacon que alterna entre estados \textbf{ATIVO}, \textbf{ESCUTA} e \textbf{BLOQUEADO}, criando janelas temporais protegidas para tráfego crítico. A implementação no framework OMNeT++/INET demonstra que o DQ-WiFi pode alcançar latência submilissegundo para alertas de emergência e limites de latência comprovadamente limitados para tráfego de controle aperiódico, superando significativamente o EDCA padrão em desempenho determinístico.

	\textbf{Palavras-chave}: V2X. Wi-Fi. QoS Determinístico. Máquina de Estados Finitos. Reserva de Banda. Comunicação Veicular.
\end{resumo}

% resumo em inglês
\begin{resumo}[Abstract]
	\SingleSpacing
	\begin{otherlanguage*}{english}
	Vehicle-to-Everything (V2X) communication requires deterministic latency guarantees for safety-critical applications, which conventional CSMA/CA-based Wi-Fi fails to provide. This thesis proposes \textbf{DQ-WiFi}, a MAC framework that combines periodic bandwidth reservation with coordinated listening to deliver deterministic QoS on legacy Wi-Fi devices. The core mechanism is a beacon-synchronized Finite State Machine (FSM) that toggles between \textbf{ACTIVE}, \textbf{LISTENING}, and \textbf{BLOCKED} states, creating protected temporal windows for critical traffic. Implementation in the OMNeT++/INET framework demonstrates that DQ-WiFi can achieve sub-millisecond latency for emergency alerts and provably bounded latency for aperiodic control traffic, significantly outperforming standard EDCA in deterministic performance.

		\textbf{Keywords}: V2X. Wi-Fi. Deterministic QoS. Finite State Machine. Bandwidth Reservation. Vehicular Communication.
	\end{otherlanguage*}
\end{resumo}
